# ForestConnect

## Giới thiệu

ForestConnect là hệ thống hỗ trợ người dân và Kiểm lâm trong công tác cập nhật diễn biến rừng bằng điện thoại thông minh.

Mục tiêu:

- Người dân báo cáo biến động rừng bằng GPS.
- Chụp ảnh hiện trường.
- Tự động tạo Polygon và tính diện tích.
- Gửi dữ liệu về máy chủ.
- Kiểm lâm tiếp nhận trên Website.
- Đồng bộ dữ liệu với QGIS.
